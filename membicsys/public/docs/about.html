The main page you are viewing shows recent profiles (people) and themes (collaborative topics). Each of these membic pages is a curated collection of membics. A membic is a link together with a reason why it is memorable.

Membic.org provides profile and theme pages that can be used for reference and for news. Each page has advanced search that makes finding resources easy, and themes may provide custom keywords to help you quickly find curated information relevant to you. Follow any page via webfeed or email to stay informed of newly added membics. Membic.org is a free service. If you would like to contribute to supporting the site, visit epinova.com.

For more information, see the blog.

Membic is an open source project where anyone can interact with issues and other aspects of development. Contact us via the project.
