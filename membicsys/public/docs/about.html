The main page you are viewing shows recent profiles (people) and themes (collaborative topics). Each of these membic pages is a curated collection of membics. A membic is a link with a reason why it is memorable.

Membic.org can be used for reference and for news. For reference, search a theme, or create your own theme for others to explore. You may embed any membic page as part of your own website. To use membic for news, follow any profile or theme. You can follow via webfeed or email.

Each membic page provides sharing, settings, and search. Many themes provide content specific search keywords to help you quickly find curated information relevant to you.

For more information on Membic, see the blog.

Membic is an open source project where anyone can interact with issues and other aspects of development. Contact us via the project.
