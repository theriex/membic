The main page you are viewing shows recent profiles (people) and themes (collaborative topics). Each of these membic pages is a curated collection of membics. A membic is a link together with a reason why it is memorable.

Membic.org provides profile and theme pages that can be used for reference and for news. Most themes provide custom keywords to help you quickly find curated information relevant to you. Follow any page via webfeed or email to stay informed.

For more information, see the blog. These blog posts describe specific features you may find helpful:

  • Why What's Memorable is So Important
  • How Membic Search Works

Membic is an open source project, and Membic.org is a free service. If you would like to contribute to supporting the site, visit epinova.com. Contact us via the project.
