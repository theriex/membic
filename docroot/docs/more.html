Connect to FGFweb directly from iTunes!

These 3 AppleScript files enable to you to characterize your tracks and automatically build situational playlists:

  • FGFwebReview is run anytime you are listening to a track. It prompts you to select appropriate keywords, provide an overall rating, decide how often the track should be selected, and add any other comments you want.
  • FGFwebPlaylist creates or updates a playlist built from keyword values and a minimum quality rating, not including anything that has been selected too recently.
  • FGFwebExport copies the playlist and source tracks out to another folder. Handy for putting music on an Android phone.

The scripts work directly within iTunes using the rating and comment fields. No modifications are required, and existing field values are preserved.

To get your FGFweb script support for iTunes:

  1. Download iTunesMac.zip
  2. Double click the downloaded iTunesMac.zip to expand it.
  3. Double click Install.scpt to open it in the AppleScript Editor. Click the big green "Run" button and quit the AppleScript Editor after it finishes.
  4. From iTunes, select a script from the scrolly looking thingy in the top menu bar.

You might find it handy to set up a keyboard shortcut for FGFwebReview:

  1. Open System Preferences | "Keyboard" | "Application Shortcuts"
  2. Click '+' to add a new shortcut
  3. Select "iTunes" as the application, FGFwebReview as the menu title, and command-; as the keyboard shortcut (iTunes already has default keybindings for most other keys).
