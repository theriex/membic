Connect to wdydfun directly from iTunes on your Mac!

If you would like to review songs within iTunes, and create playlists from keywords and ratings, this is for you:

  • WDYDFunExport copies source tracks for any playlist out to another folder.
  • WDYDFunList creates or updates the contents of any playlist built from the keywords and rating you specified.
  • WDYDFunReview sets the keywords, comments and star rating for the currently playing track.

To get your WDYDFun script support for iTunes:

  1. Download iTunesMac.zip
  2. Double click the downloaded iTunesMac.zip to expand it.
  3. Double click Install.scpt to open it in the AppleScript Editor. Click the big green "Run" button and quit the AppleScript Editor after it finishes.
  4. From iTunes, select a script from the scrolly looking thingy in the top menu bar.

You might find it handy to set up a keyboard shortcut for WDYDFunReview:

  1. Open System Preferences | "Keyboard" | "Application Shortcuts"
  2. Click '+' to add a new shortcut
  3. Select "iTunes" as the application, WDYDFunReview as the menu title, and command-; as the keyboard shortcut (iTunes already has default keybindings for most other keys).
