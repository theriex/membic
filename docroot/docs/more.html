Connect to wdydfun directly from iTunes!

There are 3 AppleScript files that enable you to quickly review tracks within iTunes, and also create playlists based on ratings and keywords:

  • WDYDFunExport copies source tracks for any playlist out to another folder.
  • WDYDFunPlaylist creates or updates the contents of any playlist built from the keywords and rating you specified.
  • WDYDFunReview updates the keywords, comments, rating and play frequency for the current track.

The review process uses the Comments field to track extra information like relevant keywords, how often a track should be automatically selected, and any additional comments you might have. The playlist creation process matches combinations of keywords, rating stars and play frequency to automatically build custom playlists for any situation. The export utility will copy the contents of any playlist to another folder and is handy for moving files to other places like an Android phone.

To get your WDYDFun script support for iTunes:

  1. Download iTunesMac.zip
  2. Double click the downloaded iTunesMac.zip to expand it.
  3. Double click Install.scpt to open it in the AppleScript Editor. Click the big green "Run" button and quit the AppleScript Editor after it finishes.
  4. From iTunes, select a script from the scrolly looking thingy in the top menu bar.

You might find it handy to set up a keyboard shortcut for WDYDFunReview:

  1. Open System Preferences | "Keyboard" | "Application Shortcuts"
  2. Click '+' to add a new shortcut
  3. Select "iTunes" as the application, WDYDFunReview as the menu title, and command-; as the keyboard shortcut (iTunes already has default keybindings for most other keys).
