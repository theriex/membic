If you have any suggestions that would make MyOpenReviews better for you, please let us know. We want this site to be awesome.

The best way to get in touch is via the GitHub issue tracker which is used for enhancement requests, bugs, and general questions. Feel free to comment on any of the existing issues or create a new issue if there isn't one there already.

If you are uncomfortable posting to the issue board, or if it doesn't seem appropriate, you can email eric. And of course you can always write a review :-)
