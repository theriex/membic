All the server calls used by the MyOpenReviews site are available for general use.

The API consists of HTTP GET or POST calls that return results formatted as JSON (XML and CSV options coming later). To make sure the API documentation stays in sync with what is actually available, the docs are maintained in source code, specifically app.yaml, where the server endpoints are defined. Take a look there to see descriptions of what is available.

For details on processing, including data structures, reference the python source for the defined script module. For example if you want to read the /login script src.py.moracct.app, you would look at myopenreviews/src/py/moracct.py for details.
