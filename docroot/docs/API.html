All the server calls used by the MyOpenReviews site are available for general use.

The API consists of HTTP GET or POST calls, with results formatted as JSON, XML, or CSV. Rather than maintain separate documentation to describe each of the possible server endpoints and how they are used, the mappings are described directly in the actual source. To see the API, check out app.yaml

If you want to delve deeper into the processing for any url/script pairing, you can substitute the dots in the associated script to read the source. For example if you want to read the /login script src.py.moracct.app, check out myopenreviews/src/py/moracct.py.

Questions, suggestions, corrections, pull requests etc. are very welcome.
